I created this app as a challenge set by myself to practice my Javascript skills. The built a simple one page user interface which allows the user to enter any date (in the format dd/mm/yyy) and then click the button, this will return a string which tells the user who the Monarch, or Monarchs of England were on the given date.

Mixed Messages

The Mixed Messages project was a challenge on the Codecademy Full-Stack Engineer pathway. The objective was to build an application which would generate a random string/fortune, by choosing randomly from different options and concatenating them together. For this project i built a simple front-end with HTML and CSS to show the output. I used Javascript to randomly select an option from three different arrays and then return these three options in a string. The app returns a year (between 1000BC - 2022AD), a fictitious event that happened that year, and the weather on that day.
